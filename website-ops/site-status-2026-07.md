## Актуальные скиллы и пайплайн Sanity (июль 2026)

Живут в репо `ai-mindset-org/ai-mindset-website`:

1. **`/custom-lab-site`** (Ракитин, 11–12.07) — самый свежий: «собери витрину
   для лабы со slug s26», доработка секций, связь с атрибутами Sanity,
   поддержка визуального редактора.
2. **`/sanity-content`** (Ракитин, 01.07) — наполнение данных Sanity агентом
   напрямую (смотрит в папку `studio`); менять структуру запрещено.
3. **Гайды вайбкод-лаб**: `docs/guides/vibecode-lab-site.md` +
   `vibecode-lab-site-guardrails.md` (не писать в Sanity, не трогать схему —
   подсовывать агенту в контекст). Вайбкод-лабы с данными Sanity кладутся в
   `src/pages/labs-custom` основного репо (webhooks), пример:
   staging.aimindset.org/labs-custom/test-internal-custom/.
4. **MD-импорт лабы** (`npm run import-lab` / кнопка в Studio;
   `create-lab-from-md.md` + свежий `common_content.md`) — работает, но
   частично вытеснен скиллами выше. Хабовский пакет `create-aim-lab-md-skill`
   построен на этом пайплайне — для наполнения предпочитать `/sanity-content`
   и `/custom-lab-site`.
5. **Кастомные блоки** в стандартной лабе (HTML в Sanity без JS) и
   **визуальный редактор текста в предпросмотре** (демо:
   loom.com/share/63a74f3968c7438c8d2dd3a75d834808). Предпросмотр до publish:
   aimindset / 0987654321; после publish — вебхук+билд 1–2 мин.
6. **Оплата на вайбкод-страницах** — инструкция Ракитина для человека/агента
   (06.06): `docs/guides/payment-embed-guide.md` в репо сайта, копия —
   `website-ops/payment-embed-guide.md`. Два способа: ссылка на
   `/pay?product=…` или попап-виджет `widget/checkout.js`
   (`data-aim-checkout` + `data-product`), автоцены `data-aim-price`, API
   `AIM.openCheckout()`. Примеры для Health:
   `reports/2026-06-06_health-pay-instruction-{pay-page,widget}.md`.
